# CAR para Todos

Plataforma web MVP para democratizar o acesso ao Cadastro Ambiental Rural (CAR), desenvolvida para o desafio haCARthon da ENAP.

## O que está implementado

- Landing page institucional com proposta, impacto e módulos
- Fluxo de autenticação simulado com perfis de produtor, analista e administrador
- Assistente inteligente com respostas rápidas sobre APP, Reserva Legal e benefícios
- Diagnóstico guiado com relatório personalizado
- Academia CAR com módulos, quiz e certificado simbólico
- Dashboard de analista e painel administrativo
- Recursos de acessibilidade com alto contraste e aumento de fonte

## Como visualizar

Abra o arquivo [index.html](index.html) em um navegador para explorar o MVP.

## Arquitetura proposta

- Frontend: Vue.js 3 (pronto para ser portado a partir do protótipo atual)
- Backend: Laravel 12
- Banco de dados: MySQL
- API REST
- Acessibilidade: navegação por teclado, alto contraste e leitura de tela
