# Design System - CAR para Todos

## 1. Princípios de design
- Clareza institucional e confiança governamental
- Acessibilidade, contraste e simplicidade
- Interface moderna, limpa e responsiva
- Linguagem objetiva e visual consistente
- Experiência amigável para diferentes perfis de usuários

## 2. Paleta de cores

### Cores primárias
- Azul governamental: #0F4C81
- Azul secundário: #1E6FB8
- Azul claro: #4F8CFF
- Azul elétrico: #6DC7FF

### Cores de apoio
- Verde sucesso: #25C26E
- Amarelo alerta: #F2B84B
- Vermelho erro: #E45757
- Cinza escuro: #11233B
- Cinza médio: #6C7A8A
- Cinza claro: #F5F8FC

### Cores de fundo
- Fundo principal: #F5F8FC
- Fundo card: #FFFFFF
- Fundo sidebar: #0D1B2E
- Fundo hover: #EEF4FF

### Uso recomendado
- Títulos e destaques: azul governamental
- Ações primárias: azul claro
- Estados de sucesso: verde
- Alertas: amarelo
- Erros: vermelho

## 3. Tipografia

### Família tipográfica
- Inter, Segoe UI, Roboto, Arial, sans-serif

### Escalas recomendadas
- H1: 32px / 700
- H2: 28px / 700
- H3: 24px / 600
- H4: 20px / 600
- H5: 16px / 600
- Body: 16px / 400
- Small: 14px / 400
- Caption: 12px / 400

### Diretrizes
- Usar texto claro e objetivo
- Priorizar títulos curtos e frases diretas
- Manter peso visual consistente
- Evitar excesso de texto em telas operacionais

## 4. Botões

### Botão primário
- Background: azul principal
- Texto: branco
- Radius: 12px
- Padding: 10px 16px
- Shadow leve

### Botão secundário
- Background: branco
- Borda: azul principal
- Texto: azul principal

### Botão sucesso
- Background: verde sucesso
- Texto: branco

### Botão perigo
- Background: vermelho erro
- Texto: branco

### Estados
- Hover: escurecer levemente ou elevar sombra
- Focus: outline azul com 2px de destaque
- Disabled: opacidade reduzida e cursor não permitido

## 5. Inputs

### Padrão visual
- Borda: 1px sólida em cinza claro
- Radius: 10px
- Padding: 10px 12px
- Background: branco

### Estados
- Focus: borda azul e sombra suave
- Error: borda vermelha
- Disabled: fundo cinza claro e texto desativado

## 6. Cards

### Padrão geral
- Background: branco
- Radius: 16px
- Shadow: 0 8px 24px rgba(17, 35, 59, 0.06)
- Borda sutil em cinza claro

### Uso recomendado
- Resumos de dados
- Listas de itens
- Blocos de conteúdo
- Paineis de destaque

## 7. Sidebar

### Visual
- Background: azul escuro institucional
- Itens com espaçamento consistente
- Ícones alinhados à esquerda
- Destaque para item ativo

### Comportamento
- Item ativo: background azul claro com contraste alto
- Hover: background mais claro
- Responsiva: recolhível em telas pequenas

## 8. Navbar

### Visual
- Fundo branco ou translúcido com borda sutil
- Logo à esquerda
- Menu central ou à direita
- Botão de ação principal destacado

### Comportamento
- Fixa no topo em telas maiores
- Menu responsivo em telas pequenas

## 9. Footer

### Visual
- Fundo azul escuro ou cinza profundo
- Texto claro e legível
- Seções de contato, links e acessibilidade

## 10. Alertas

### Tipos
- Success: verde com texto escuro e ícone de confirmação
- Warning: amarelo com texto escuro
- Error: vermelho com texto claro
- Info: azul claro com texto escuro

## 11. Badges

### Uso recomendável
- Status ativo, pendente, concluído, novo
- Backgrounds coloridos com contraste alto
- Radius arredondado

## 12. Modais

### Padrão visual
- Fundo escurecido de overlay
- Card central com radius 20px
- Cabeçalho, corpo e rodapé bem delimitados
- Botões alinhados no rodapé

## 13. Tabelas

### Visual
- Cabeçalhos com fundo suave
- Linhas separadas por borda leve
- Hover suave nas linhas
- Boa leitura em telas pequenas

## 14. Formulários

### Estrutura recomendada
- Grupo de campo com label acima do input
- Mensagens de erro abaixo do campo
- Botões de ação no fim do formulário
- Espaçamento consistente entre etapas

## 15. Ícones

### Biblioteca recomendada
- Bootstrap Icons

### Uso
- Ícones simples, limpos e consistentes
- Não exagerar no número de ícones por tela
- Sempre com função clara

## 16. Estados de interação

### Hover
- Elevation leve ou mudança de cor
- Cursor pointer em elementos clicáveis

### Focus
- Outline azul claro com 2px
- Visível em teclado e mouse

### Disabled
- Sem interação
- Redução de contraste e opacidade

## 17. Diretrizes de acessibilidade
- Contraste adequado entre texto e fundo
- Componentes com foco visível
- Labels claros em formulários
- Botões com tamanho mínimo confortável
- Suporte para leitura de tela e navegação por teclado

## 18. Aplicação prática no projeto
Este design system deve ser aplicado em:
- Landing page
- Dashboard
- Assistente
- Diagnóstico
- Academia CAR
- Mapa de benefícios
- Painel administrativo
- Formulários e modais
